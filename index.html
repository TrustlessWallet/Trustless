Trustless

Trustless is a fully open-source, non-custodial, privacy-focused, Bitcoin-only mobile wallet. It is very minimalist and easy to use, yet very functional.

View Source Code (GPLv3)

  • Bitcoin wallet

    Import or create new wallets, track balances, send and receive Bitcoin.

  • Address tracker

    Track any Bitcoin address balance just by entering it. No private keys required.

  • Receive address change

    Have 20 unused receive addresses at all times to choose from. This preserves privacy by preventing address reuse.

  • Balance breakdown

    See actual UTXOs you own by clicking on your total wallet balance.

  • Coin control

    Choose specifically which UTXOs to use for a transaction.

  • BIP44 change address management

    All change addresses are derived from a 1/n chain and are only used once.

  • Custom node connection

    Connect to your own node via Electrum.

  • Network switch

    Use testnet network for transaction testing and development.
