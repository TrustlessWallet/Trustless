Trustless

[cite_start]Trustless is a fully open-source, Bitcoin-only mobile wallet[cite: 5]. [cite_start]It is very minimalist and easy to use, yet very functional[cite: 5].

[cite_start]View Source Code (GPLv3) [cite: 5]

  • Bitcoin wallet

    [cite_start]

    Import or create new wallets, track balances, send and receive Bitcoin[cite: 5].

  • Address tracker

    Track any Bitcoin address balance just by entering it. [cite_start]No private keys required[cite: 5].

  • Receive address change

    Have 20 unused receive addresses at all times to choose from. [cite_start]This is a privacy feature, because address reuse links payments together[cite: 5].

  • Balance breakdown

    [cite_start]

    See actual UTXOs you own by clicking on your total wallet balance[cite: 5].

  • Coin control

    [cite_start]

    Choose which UTXOs to use for a transaction[cite: 5].

  • BIP44 change address management

    [cite_start]

    Following best privacy practices, all change addresses are derived from a 1/n chain and are only used once[cite: 5].

  • Custom node connection
